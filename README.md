# Installation Instructions

## Installing the Custom Seating Blender Addon (.py file)

Follow these steps to install the addon directly from the downloaded Python file.

1. Download the Addon File  
    `cushion_processing_addon.py`  
    Save it somewhere easy to locate like `C:\BlenderAddon`, or a folder in your profile.

2. Open Blender  
    Launch Blender.

3. Open the Add‑ons Preferences  
    Navigate to:  
    Edit → Preferences… → Add-ons

4. Install the Addon  
    Inside the Add‑ons window:      
    Click Install… (top‑right)  
    Browse to the downloaded file cushion_processing_addon.py  
    Select it and click Install Add-on

5. Enable the Addon
    Search for `Cushion Processing Tools`
    Tick the checkbox to enable the addon

6. Accessing the Tools in Blender  
    Press N to open the right‑hand sidebar  
    Click the “Cushion Processing” tab  
    The tool panel “Cushion Processing Tools” will appear

    This is where you will find options for scan alignment, flipping, and any future tools.

7. Updating the Addon  
    Go to Edit → Preferences → Add-ons  
    Locate the addon  
    Click the small arrow next to the addon name  
    Choose Remove  
    Reinstall by repeating the steps 1-5 above with the new .py file
